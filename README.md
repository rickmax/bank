# Bankmax

## The Bankmax is a application to test bank operations like deposit, transfer and withdraw.

### Ruby version
    2.6.6
### Rails version
    5.2.4
### System dependencies
    pstgresql
### Database creation
    rails db:create db:migrate
### How to run the test suite
    bundle exec rspec
### Deployment instructions
    git push heroku master
